# 💼 Job Hunt Pro

A responsive and user-friendly Job Portal web application built using **HTML, CSS, and JavaScript**. The application allows users to browse jobs, search and filter listings, save their favorite jobs, and explore hiring companies through a clean and modern interface.

---

## 🚀 Features

- 🏠 Responsive Home Page
- 🔍 Search jobs by title or company
- 🎯 Filter jobs by category
- 💾 Save and remove favorite jobs
- 🏢 Companies page
- 📱 Mobile-friendly design
- ⚡ Interactive UI using JavaScript
- 💻 Clean and modern layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
- Font Awesome
- Google Fonts

## ▶️ How to Run the Project

1. Download or clone the repository.

```
git clone https://github.com/your-username/job-hunt-pro.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

No additional installation is required.
